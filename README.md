## Setting Up \"shop-intel\" Repository\n\n1. **Clone the Repository:**\n   ```bash\n   git clone git@github.com:arkamaldeen/shop-intel.git\n   ```\n\n2. **Navigate to the Repository Directory:**\n   ```bash\n   cd shop-intel\n   ```\n\n3. **Create a Virtual Environment:**\n   ```bash\n   python -m venv .venv\n   ```\n\n4. **Activate the Virtual Environment:**\n   ```bash\n   .venv\\scripts\\activate\n   ```\n\n5. **Install Dependencies in the Virtual Environment:**\n   ```bash\n   pip install -r requirements.txt\n   ```\n\n6. **Install Docker and Run Docker Desktop:**\n   ```bash\n   docker pull qdrant/qdrant\n   ```\n\n7. **Run Qdrant Docker:**\n   ```bash\n   docker run -p 6333:6333 -v .:/qdrant/storage qdrant/qdrant\n   ```\n\n8. **Run Python File:**\n   ```bash\n   python app.py\n   ```\n\nThis set of instructions guides you through setting up and running the \"shop-intel\" repository, including cloning the repository, setting up a virtual environment, installing dependencies, running Docker, and executing the Python file.
